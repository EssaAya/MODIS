### Installation

As of 10 January 2017, **MODIS** is finally available from [CRAN](https://cran.r-project.org/package=devtools) and can be installed via 


```S
install.packages("MODIS")
```


To install the development version of the package, first install [devtools](https://cran.r-project.org/package=devtools) and subsequently run

```S
devtools::install_github("MatMatt/MODIS", ref = "develop")
```
